HOLA MUNDO!

Veamos cómo configurar las fuentes de un contenido, disponible en un elemento HTML. Puedes establecer las siguientes propiedades de fuente de un elemento: La propiedad font-family se utiliza para cambiar la cara de una fuente. La propiedad de font-style se usa para hacer una fuente en cursiva u oblicua. La propiedad font-variant se utiliza para crear un efecto de versalitas. La propiedad de font-weight se utiliza para aumentar o disminuir la negrita o la luz de una fuente. La propiedad de font-size se utiliza para aumentar o disminuir el tamaño de una fuente. La propiedad de font se utiliza como forma abreviada para especificar otras propiedades de fuente.
